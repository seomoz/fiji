
                     Fiji Phonebook example release notes


Version 1.0.0-rc4
  * DOCS-26. Update bulk importing to use FijiMR; deprecate MapReduce
      examples. Check out the "Fiji Music" example project for a
      complete FijiMR example application.
  * SCHEMA-133. Update usage to match API from KjiiSchema 1.0.0-rc4
      (use FijiDataRequestBuilders instead of mutable FijiDataRequests.)

Version 1.0.0-rc3
  * SCHEMA-50. Update usage to match API from FijiSchema 1.0.0-rc3
      (release() and retain() methods for Fiji objects)
  * SCHEMA-97. Changed instances of Fiji.open(...) to Fiji.Factory.open(...).

Version 1.0.0-rc2
  * PHONE-7. Bugfix: Set zip code to 'int' in layout.
  * PHONE-6. Use deprecated FijiSchema MapReduce classes.
      These classes are marked as deprecated, because they will be
      replaced by the Fiji MapReduce library. Stay tuned for that
      release.
  * SCHEMA-59. Declare an explicit dependency on the CDH4 platform module.
  * PHONE-2. Support reading console input from Eclipse.

Version 1.0.0-rc1
  * First release candidate.
